**Modern Portfolio** is a high-performance business card website designed with a focus on speed, modern UX, and clean architecture. The project demonstrates skills in creating responsive interfaces and working with the latest tech stack for 2026.

## Tech stack

The project is built on **Vite**, which provides instant build and hot reload (HMR).

* **Frontend:** [React.js](https://reactjs.org/) (Functional Components, Hooks)
* **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) (Next-gen CSS engine)
* **Icons:** [React Icons](https://react-icons.github.io/react-icons/)

## Peculiarities

- **Performance:** Optimized build with Lighthouse 95+ metrics.
- **Design:** Responsive layout, Retina display support, and smooth animation.
- **Architecture:** A component-based approach that makes scaling easier.
- **Mobile First:** The interface is fully adapted for smartphones and tablets.

## Quick start

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/v01dedknight/showcase.git](https://github.com/v01dedknight/showcase.git)
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
