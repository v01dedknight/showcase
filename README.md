**Showcase** is a high-performance developer portfolio designed with a focus on modern UX, interactivity, and clean component architecture. The project showcases relevant approaches to web development in 2026.

## Tech stack

The project is built on **Vite**, which provides instant build and hot reload (HMR).

* **Framework:** React.js (Functional Components, Hooks)
* **Build Tool:** Vite (Instant HMR & Fast Build)
* **Styling:** Tailwind CSS v4 (Next-gen CSS engine)
* **Animations:** Framer Motion & CSS Transitions
* **Icons:** React Icons (Lucide, FontAwesome, HeroIcons)

## Peculiarities

- **Dynamic Navigation:** Informational Navbar with real-time local time and availability status.
- **Interactive Showcase:** Project slider with a progress bar and detailed modal windows for each project.
- **Glassmorphism Design:** Use of modern blur effects, gradients, and floating interface elements.
- **Modern UX:** Smooth scrolling, interactive button states, and optimized navigation.
- **Performance:** High Core Web Vitals scores thanks to resource optimization and the use of Vite.

## Features

- **Project Modals:** Detailed descriptions of each project, technology stack, and direct links to demos.
- **Smooth Scroll:** Programmatic navigation to the contacts section.
- **Responsive:** Fully adaptive to any screen resolution, from mobile devices to 4K monitors.

## Quick start

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/v01dedknight/showcase.git](https://github.com/v01dedknight/showcase.git)
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
